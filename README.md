## Pages

### Home
Landing page with a hero section showcasing the designer's name and CTA buttons.

### Portfolio
Grid layout displaying design projects with images, descriptions, and tags.

### About
Bio section with designer information, skills, and expertise areas.

### Contact
Contact information and a contact form for inquiries.

## Customization

1. **Update Portfolio Projects**: Edit the `projects` array in `src/pages/Portfolio.js`
2. **Change Colors**: Modify the gradient colors in CSS files (currently using purple/blue gradient)
3. **Add Contact Information**: Update contact details in `src/pages/Contact.js`
4. **Replace Images**: Update placeholder image URLs with your actual design images
